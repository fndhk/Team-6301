3. 실행한다

5. 잘못 생성한 노트는 누르면 해당 노트가 사라진다.
6. 상단의 버튼과 스크롤바로 음악 위치 조정 가능
7. 노트 맵 드래그하거나 마우스 휠로 움직일 수 있다.
8. 완성했으면 상단의 Export 버튼을 누른다.
9. 폴더가 열리며 해당 비트맵의 json 파일이 생성됐을 것이다.
10. 실제 게임(defence) 유니티를 켜서 Resources 폴더에 Beatmaps 폴더에 json 파일을 붙여넣는다.(meta파일은 필요 없음)
11. 유니티창 맨 위에 Tools를 누른다 
12. Beatmap Importer를 선택한다
13. 아까 가져온 json 파일을 넣고 비트맵 이름을 지정하고 Import Beatmap 버튼을 누른다.
14. StageData에 있는 스테이지 데이터 Beatmap에 완성된 비트맵을 집어넣는다.
