✅ 1순위: 신규 핵심 아이템 제작 (좌우 타워 활성화)
가장 먼저 게임 플레이에 큰 변화를 줄 핵심 아이템부터 제작합니다.


[ ] ActivateTowerEffect.cs 아이템 효과 스크립트 제작

ItemEffect를 상속받아 새로운 효과를 만듭니다.

이 효과는 TowerSlot에 있는 타워를 활성화시키는 역할을 합니다.

[ ] 신규 아이템 데이터(ItemData) 에셋 생성

'드럼 활성화', '심벌즈 활성화' 같은 이름의 아이템 데이터를 만듭니다.

새로 만든 ActivateTowerEffect를 연결하고, 아이콘 이미지를 지정합니다.

[ ] 아이템 획득 경로 추가

적이 낮은 확률로 드롭하도록 Enemy.cs의 lootTable에 신규 아이템을 추가합니다.

🛠️ 2순위: 영구 성장 시스템 구현
플레이어가 게임을 계속해서 즐길 동기를 부여하는 시스템을 완성합니다.

[ ] 강화 전용 씬(EnhancementScene) 제작

보유 재료, 강화 버튼, 능력치 정보, 뒤로가기 버튼 등 UI를 배치합니다.

[ ] EnhancementUI.cs 스크립트 작성

UI와 데이터를 연동합니다.

버튼 클릭 시 GameData의 permanentAtkBonus, permanentCoreHpBonus 값을 올리고 enhancementMaterials를 차감한 뒤, 게임을 저장하는 기능을 구현합니다.

[ ] MainMenu 또는 StageSelect 씬에 '강화' 버튼 추가

EnhancementScene으로 이동하는 버튼을 만듭니다.

✨ 3순위: 폴리싱 (사운드 & 시각 효과)
게임의 손맛과 완성도를 끌어올립니다.

[ ] 사운드 이펙트 추가

필수: UI 클릭음, 아이템 획득/사용음, 적 사망음

권장: 웨이브 시작/종료 알림음

[ ] 시각 효과 (파티클) 추가

필수: 투사체가 적에게 맞았을 때 터지는 임팩트 이펙트

권장: 적 사망 시 사라지는 이펙트, 아이템 사용 시 타워 주변 이펙트

⚖️ 4순위: 밸런싱 및 콘텐츠 제작
모든 기능이 완성된 후, 게임을 '재미있게' 만드는 최종 단계입니다.

[ ] 게임 밸런싱

스테이지별 적 능력치(체력, 공격력 등)를 조절하여 난이 곡선을 설계합니다.

타워의 기본 공격력과 판정별 데미지 배율을 최종 조절합니다.

강화 재료 획득량과 강화 비용을 조절하여 성장 속도를 결정합니다.

[ ] 최종 채보 제작

게임에 사용할 실제 음악에 맞춰 BeatmapData 에셋을 제작하고 테스트합니다.
